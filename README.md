# pipline — Data Science Pipeline for Analytics

Этот репозиторий содержит инструменты и скрипты для построения воспроизводимых пайплайнов обработки и анализа данных.

## Структура проекта

- `data/` — входные и выходные данные (raw, processed, external)
- `notebooks/` — Jupyter-ноутбуки для исследовательского анализа (EDA)
- `src/` — исходный код пайплайна (ETL-логика, утилиты)
- `tests/` — тесты

## Установка

```bash
pip install -r requirements.txt
```

## Запуск пайплайна

```bash
python -m src.pipeline
```

## Структура данных

```
data/
├── raw/        — необработанные данные (например, CSV, JSON)
├── processed/  — очищенные и подготовленные датасеты
└── external/   — внешние данные (справочники, геоданные и т.д.)
```

## Примечания

- Конфигурация хранится в `src/config.py`.
- Используйте `src/utils.py` для общих функций (логирование, валидация и т.д.).
- Начните с ноутбука `notebooks/explore_data.ipynb`.
